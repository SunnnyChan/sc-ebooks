```md
What is BPF?
Berkeley Packet Filter (BPF) 是一个内核执行引擎，用于处理虚拟指令集，
最近进行了扩展（也称为eBPF），以提供扩展内核功能的安全方法。

Extended BPF 是内置的Linux内核技术，自3.18起已部分添加。要使用本书中的工具，至少需要Linux 4.9。
```

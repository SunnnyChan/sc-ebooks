## 简介
```md
相较于 GoF 的设计模式一书，本书介绍的模式涉及多个层次的抽象：从高层的架构到设计模式，再到低层的成例（idiom）。
还专注于与OOP无关的问题。

本书总体目标帮助读者在软件架构的大背景下更好的应用模式解决问题。
```
